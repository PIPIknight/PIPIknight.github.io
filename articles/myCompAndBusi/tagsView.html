堆叠拖拽 #

​ 关于堆叠效果，本来是用 绝对定位 + z-index 控制的，结果在实际拖拽的时候出现了大问题（具体什么问题可以亲自试试），然后才想起来可以用 margin负值 + z-index 处理这种堆叠，也免去了拖拽时繁琐的计算。

​ 下面展示相关代码

vue
<template>
    	<!-- …… -->
        <div class="tab-names" ref="tabBar">
            <div class="tab-name-group" ref="tabBarIn" style="position:relative">
                <div v-for="item, index in currentList" :key="item.path" :sortName="item.path" class="tab-node"
                    @click="toggleTag(item.path)" :class="{ 'is-active': activeTag == item.path }"
                    @contextmenu.prevent="onContextmenu(item, $event)" :style="{ 'z-index': item.zIndex }">
                    <div class="tab-label">
                        <span style="margin-right:8px">{{ item.title }}</span>
                        <el-icon :size="16" color="" @click.stop="closeTag(item.path)"
                            :class="{ 'is-last': currentList.length <= 1 }">
                            <ele-CloseBold />
                        </el-icon>
                    </div>
                </div>
            </div>
        </div>
        <!-- …… -->
    </div>
</template>
<script lang="ts">
    // ……
    
    // 初始化tagsView列表，主要的作用是设置 z-index
    const initTagsView = () => {
        // if (tagsView.value.visitedViews.length == 0) return currentList.value = []
        let cacheList: any[] = []
        tagsView.value.visitedViews.forEach((item, index) => {
            const zIndex = 10 + tagsView.value.visitedViews.length - index
            cacheList.push({ ...listMap[item], zIndex })
        })
        currentList.value = [...cacheList]
        activeTag.value = tagsView.value.currentTag
        router.push(tagsView.value.currentTag)
    }
    
    // 进行监听，同时重置 z-index
    watch(() => tagsView, (val) => {
        initTagsView()
    }, { deep: true })
    
    // sortable拖拽
    const initSort = () => {
        tabSort.value = new Sortable(tabBarIn.value, {
            animation: 150,
            forceFallback: true,
            onEnd: (e: any) => {
                const nodeList = e.to.getElementsByClassName('tab-node')
                const newSort = []
                for (let i = 0; i < nodeList.length; i++) {
                    newSort.push(nodeList[i].getAttribute('sortName'))
                }
                useTagsView().resetTagsViewRoute(newSort)
            }
        })
    }

</script>

左右移动 #

​ 之前左右移动都是靠 el-scroller 解决的，不过自己写左右移动也很简单，只要理解，外部盒子是处于 隐藏溢出 状态，移动内部盒子的绝对定位，实现左右移动就行！

js
// 左右移动
const moveTab = (displacement: number) => {
    if (tabBar.value.offsetWidth > tabBarIn.value.offsetWidth) return
    // 内外差值为右边界
    const max = tabBar.value.offsetWidth - tabBarIn.value.offsetWidth - 32
    // 控制边界
    if (displacement > 0) {
        let move = tabBarIn.value.offsetLeft + displacement
        if (move > 0) move = 0
        tabBarIn.value.style.left = move + 'px'
    } else {
        let move = tabBarIn.value.offsetLeft + displacement
        if (move < max) move = max
        tabBarIn.value.style.left = move + 'px'
    }
}

右击展开操作弹窗 #

​ 涉及到一个鼠标右击事件 contextmenu event

​ contextmenu 事件会在用户尝试打开上下文菜单时被触发。该事件通常在鼠标点击右键或者按下键盘上的菜单键时被触发，如果使用菜单键，该上下文菜单会被展示 到所聚焦元素的左下角，但是如果该元素是一棵 DOM 树的话，上下文菜单便会展示在当前这一行的左下角。

​ 任何没有被禁用的鼠标右击事件 (通过调用事件的 preventDefault() 方法) 将会使得 contextmenu 事件在目标元素上被触发。

​ 首先，在鼠标右击时，需要获取鼠标右击的 x,y 位置，以及右击的tab内容。

vue
<!-- contextmenu.prevent 事件传递的数据 -->

        <span style="margin-right:8px">{{ item.title }}</span>

</div>
js
// 右键点击时：传 x,y 坐标值，以及tab的内容 传到子组件中（props）
const onContextmenu = (v: any, e: any) => {
    const { clientX, clientY } = e;
    dropdown.x = clientX;
    dropdown.y = clientY;
    contextmenuRef.value.openContextmenu(v);
};

子组件接收数据并处理

js
// 父级传过来的坐标 x,y 值
const dropdowns = computed(() => {
    // 117 为 `Dropdown 下拉菜单` 的宽度
    if (props.dropdown.x + 117 > document.documentElement.clientWidth) {
        return {
            x: document.documentElement.clientWidth - 117 - 5,
            y: props.dropdown.y,
        };
    } else {
        return props.dropdown;
    }
});

// 点击命令，将上边传入的 v 即tab信息，及要执行的命令告知父组件！
const onCurrentContextmenuClick = (contextMenuClickId: number) => {
    emit('currentContextmenuClick', Object.assign({}, { contextMenuClickId }, state.item));
};
